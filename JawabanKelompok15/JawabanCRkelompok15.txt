code smell divergent change mengacu pada perubahan yang dilakukan pada kode tidak ada kaitan tetapi lebih kepada kepentingan yang tidak terkait satu sama lain.kita menghapus class shoppingCart dan priceCalculator. selanjutnya membuat class baru yaitu checkoutHandler untuk bertanggung jawab mengelola pesanan. karena pada class shoppingCart dan priceCalculator menangani beberapa pengelolaan yang membuat code smell divergent change. jadi setiap tanggung jawab harus diurus oleh class terpisah.

code smell parallel hierarchies terjadi ketika ada dua atau lebih inheritance class yang memiliki hubungan serupa seperti class yang mengidentifikasikan jenis sama tetapi inheritance class tersebut berjalan pararel dab terpisah satu sama lain. jadi penyelesaian soal tersebut adalah menggabungkan inheritance class menjadi satu class inheritance yang terintegrasi
