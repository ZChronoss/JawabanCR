code smell switch statement yaitu ada banyak kondisi pada case statement yang mengakibatkan duplicade kode. seperti masalah tersebut pada 'printPilihanOperasi' penggunaan if else(looping yang terlalu panjang. solusinya yaitu menggunakan polimorfisme untuk menghilangkan if else(looping) dan menggunakan method 'pilihanOperasi()' setelah itu diimplementasikan kepada masing-masing subclass. 

code smell  temporary field yaitu ketika objek memiliki variabel instance yang digunakan semetanra kemudian tidak diperlukan lagi. biasanya variabel instance hanya digunakan untuk penyimpanan sementara. selanjutnya kita menghapus  PriceCalculator dan membuat class baru yaitu PriceCalculator. pada code tersebut terdapat totalPrice sebagai constructor yang di gunakan pada getTotalPrice namun variabel etrsebut hanya digunakan untuk sementara. solusinya memindahkan totalPrice kedalam getTotalPrice sehingga menjadi variabel lokaldan digunakan pada method tersebut.
